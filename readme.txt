1- Download and install Google Chrome if you do not have.
2- Check chrome version and download chromedriver with same version, put into same directory where program (scraper.py) is.
3- Download and install python3.8 and add the install directory of python to environment path.
4- Open new cmd (may be you need to run as administrator)
5- run the command and cd to program directory, (the file requirements.txt should be in there)
    > pip install -r requirements.txt
6- now run the program using command
    > python scraper.py <path to output json file>
7- You can use CTRL + C if you want to stop program without finishing with all URLs. You will get the data that was scraped till that time.
